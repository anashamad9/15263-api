15263 Offers API Auto Sync

A public JSON feed for offer names, coupon codes, affiliate IDs, discounts, and GEO values. It reads only columns B-F from the Google Sheet and refreshes through GitHub Actions every 15 minutes.

What this API returns

One JSON file containing the offers assigned to affiliate ID 15263. Each offer includes its name, coupon or tracking code, affiliate ID, discount terms, and GEO.

The JSON file is regenerated from Google Sheets on a schedule, so updates in the sheet can flow into the public endpoint without manual editing.

Quick test

curl api/offers.json
const res = await fetch("api/offers.json");
const data = await res.json();
console.log(data.offers);

JavaScript

fetch("api/offers.json")
  .then(res => res.json())
  .then(data => {
    data.offers.forEach(offer => {
      console.log(offer.offer_name, offer.code, offer.discount);
    });
  });

Python

import requests

response = requests.get("https://your-domain.example/api/offers.json")
data = response.json()

for offer in data["offers"]:
    print(offer["offer_name"], offer["code"], offer["geo"])
